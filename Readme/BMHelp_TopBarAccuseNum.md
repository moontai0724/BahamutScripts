巴哈姆特板站務用工具
安裝後可於巴哈全站內頁天顯示檢舉數提醒，點擊圖示後可以列出各板有多少檢舉數，如此一來就可以在各個看板瀏覽時都看見有檢舉。
列表中皆為連結可點擊，點擊後導向到後台。
列表使用巴哈頂端按鈕樣式，因此應無違和感。
雖然這並不是非常隱密的 code，但也請各位板站務勿將此腳本給非管理人員。

現在全站都可以在內頁天顯示檢舉數了！
